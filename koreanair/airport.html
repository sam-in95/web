  • 국제선 - 국제선

  • ※ 제1여객터미널 운항 : 아시아나항공, 저비용항공, 기타 외국항공

    ※ 제2여객터미널 운항(총11개) : 대한항공, 델타항공, 에어프랑스,
    -KLM 네델란드항공, 아에로멕시코, 알리탈리아, 중화항공,
    -가루다인도네시아, 샤먼항공, 체코항공, 아에로플로트

    A. 제2여객터미널 내 환승 시 : 제2여객터미널 도착 → 2층 환승보안 검색대 → 3층 출국장

    B. 제2여객터미널 → 탑승동 또는 제1여객터미널 환승 시
    - 탑승동 환승(Concourse)
    1) 인천공항 제2여객터미널 도착
    2) 2층 1-132 Transfer 표지판을 따라 이동
    3) 지하1층 IAT(셔틀트레인) 탑승 후 탑승동 하차
    4) 환승 게이트 (101-132 Transfer 표지판) 통과
    5) 탑승동 2층 환승보안검색대 → 3층 출국장

    - 제1여객터미널 환승
    1) 인천공항 제2여객터미널 도착
    2) 2층 1-132 Transfer 표지판을 따라 이동
    3) 지하1층 IAT(셔틀트레인) 탑승 후 탑승동 하차
    4) 환승 게이트(1-50 Transfer 표지판) 통과
    5) IAT 탑승(탑승동 - 제1여객터미널)
    6) 제1여객터미널 2층 환승보안검색대 → 3층 출국장

    C. 최소 연결시
    - 제2여객터미널 -> 제2여객터미널 : 45분
    - 제2여객터미널 -> 제1여객터미널 : 1시간30분
    - 제1여객터미널 -> 제1여객터미널 : 1시간10분
    - 제1여객터미널 -> 제2여객터미널 : 1시간30분

  • 국제선 - 국내선

  • ※ 환승경로

    A. 인천공항 → 김포공항 : 입국심사 → 수하물 수취 → 세관 검사 → 김포공항 이동
    * 인천공항-김포공항 간 무료셔틀은 미운행

    B. 인천공항 제2여객터미널 → 인천공항 제1여객터미널 : 입국심사 → 수하물 수취 → 세관 검사
    → 제1여객터미널이동
    (공항철도 또는 셔틀버스) → 1층 동편 국내선 수속 카운터
    *대한항공은 국내선 미운항

  • 국내선 - 국제선

  • ※ 환승경로

    A. 인천공항 제1여객터미널 국내선 도착 → 수하물 수취 (1층)
    → 제2여객터미널 이동(공항철도 또는 셔틀버스) - 탑승수속 (3층)
